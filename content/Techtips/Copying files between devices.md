## LocalSend 

### What's it for?
At home I have a PC, Android Phone and iPad.  I want to quickly copy files and text between them.  They are all on the same home network.  

Yes I can use a 'copy twice' via a middle-system (Dropbox, Google Drive, iCloud, my Network storage box, USB stick, 💾...) but let's go **direct** between the device.  This is where *LocalSend* can help

![alt text](https://play-lh.googleusercontent.com/6HhvWTq1wUYcSPs4LrulVY8bDz3hHalz_EgYtU-SIG1fZSGfbna41TEQa8lY38zYZdo=w2560-h1440-rw)

It's not just the whole file (image, PDF, video...) you can just select some text and copy that to the other device.

### Link and Usage
LocalSend is an "[Open source](https://opensource.com/resources/what-open-source) and cross-platform file sharing for everyone" and available [here](https://localsend.org/) which has links to Apple and Android app stores too.  It's free too. 

Just install it from the above [page](https://localsend.org/).

Each device is given a fun name "Fast Orange".  You 'see' that as the target system (see below).

> [!info] Same network only
> To restate: this only works between devices on the **same** network, eg. at home. It won't work 'over the Internet'  

#### On Windows (and Mac, I assume)
Open the app to **Send** or **Receive**. Drag and drop files etc. 

#### On Mobile (Android, Apple)
To **Send**, just use the standard "Share" option.
 
To **Receive** have the App open, mine defaults to being in Receive mode

### Send away...

You will see the available peer devices with the Funny Name and usually the platform/type (ie if On Windows, you will see your iPad as 'target').  Just 'send' the file - or copied text -  that to device

==Important==: you must go to that **Receiving device** (iPad, here) and accept the 'receive' request. 

As I said, you can select text on your phone and send the just the text to your target device. 

I think you can send to more than one device at once!

## Via the Configuration Screen

You can set where - which folder - the Received file goes.

Quick Save (no prompting, just saves) is available, but OFF by default. Comes with a warning when you enable it.

Can set a PIN if you want.  

Lots of other choices.
