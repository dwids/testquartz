Got this, my first PDA-phone and am quite happy with it. I&#8217;ve previously had 2 Palm Pilots, but these weren&#8217;t phones. However a few things that the Palm&#8217;s had, but I can&#8217;t (yet) find on this Windows-based p3600i are:
- A database manager 
- Categories on Items
- The ability to add my own Applications to the main menu (aka Today)

The DB manager was, to be fair, an add-on application on the Palms. But it did enable me to set up some good DBs, like Loans and Items-To-Get.  I&#8217;m sure there&#8217;s an SQL or mySQL port to Windows Mobile 6.
