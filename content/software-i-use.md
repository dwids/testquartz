I only use Windows because I have to. I&#8217;m gradually moving off it to Linux, including running non-Microsoft software under Windows. This includes &#8216;free&#8217; (Open Source) applications. Most of these below fall into this category.

Browser:  Mozilla [Firefox](http://www.mozilla.org/products/firefox/) with the excellent [Googlebar ](http://googlebar.mozdev.org/screenshots.html) extension.

Email:  Mozilla [Thunderbird](http://www.mozilla.org/products/thunderbird/)

Music Librarian:  I used to use Media Monkey, but have now moved to the Apple&#8217;s [iTunes](http://www.apple.com/itunes/download/)

File downloads (smaller files): [Limewire](http://www.limewire.org/)

and for larger files:  the Bittorrent network via the [Azureus ](http://sourceforge.net/projects/azureus/)client program

For document creation: The brilliant [Open Office](http://www.openoffice.org/) can not only read MS Office formats, but create them too. Word. Powerpoint. Excel.  More importantly, it&#8217;s own formats are future-proof; XML text. Unlike the closed, binary formats of Office.
