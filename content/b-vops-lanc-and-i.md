> &#8220;Also I currently use VirtualdubMPEG2, resized via inbuilt filter to 576&#215;324 Lanc, 2 pass, AS @ L5, ( B-VOPS 2 / 1.5 / 1.00) with [ Adap Q, Interlaced Enc, QPEL, GMC] all off, if that helps&#8221;

.
**If that helps?**  I sure hope the target speaks in abbrev&#8217;s and acro&#8217;s 🙂 I&#8217;m seeking advice and &#8216;tuning tips&#8217; for some video conversion.

- Lanc = Lanczos resizing.  
- AS @ L5 is an MPEG-4 profile, which defines the characteristics of the target (playback) device
- B-VOPS,  Adaptive Quantization,  Interlaced Encoding, Quarter Pixel and Global Motion Compensation  are all options for encoding.  

A VOP is a Video Object  Plane, sort of like a frame. B means Bidirectional; this frame&#8217;s compression is influence by the frames before and after it.
